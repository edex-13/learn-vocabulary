### 🤖 Quick guide to download the project

1.  **Clone the repository.**

    In case you haven't: 

    ```sh
    git clone git@github.com:edex-13/learn-vocabulary.git
    cd learn-vocabulary
    ```
    Ya tienes el proyecto ❤
2.  **Install dependencies.**

    ```sh
    npm install
    ```
3.  **Run the application.**
  
      ```sh
      npm start
      ```
4.  **Enjoy!**
